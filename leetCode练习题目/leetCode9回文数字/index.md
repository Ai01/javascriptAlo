# 回文数字

## 题目

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

```
输入: 121
输出: true
```

示例 2:

```
输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
```

示例 3:

```
输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。
```


## 解答

### 方法1

将数字转换为字符串。然后利用字符串的回文判断方法。

### 方法2

如果数字无法转换为字符串。那么需要获取数字的反读数字。

用123举例。
```
y = 0;

y = y * 10 + num(123不断右移的各位数字) 

y = 3;
y = 30 + 2;
y = 320 + 1;
y = 321 + 0;
```

然后比较两者是否相同就行了。
