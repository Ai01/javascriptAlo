# 括号的合法排列

## 题目

这个题目说的是，给你 n 对括号，你要返回这 n 对括号的所有合法排列方式。

```
比如说，n 等于 3 时，合法的排列有 5 个：

((()))
(()())
(())()
()(())
()()()
```

## 方法

### 方法1

#### 思路

递归的解决问题。

```
假设leftNum为左括号剩余的数量。rightNum为右括号剩余的数量

1. 当leftNum大于0的时候。可以添加左括号
2. 当rightNum>leftNum的时候可以添加右括号

根据这两个原则写出递归函数 
```

#### 复杂度

1. 时间复杂度：卡特兰数 
2. 空间复杂度：卡特兰数
